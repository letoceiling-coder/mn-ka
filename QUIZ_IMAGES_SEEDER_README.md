# Quiz Images Seeder - Инструкция

## Описание

Сидер `QuizImagesSeeder` создает изображения для квиза в медиа-библиотеке. Он может:
- Копировать изображения из старого проекта (если путь указан)
- Создавать placeholder изображения (если исходные файлы не найдены)

## Использование

### Локальная разработка (Windows)

1. Если у вас есть старый проект по пути `C:\OSPanel\domains\lagom`, сидер автоматически попытается найти файлы там.

2. Для указания другого пути, добавьте в `.env`:
   ```env
   OLD_PROJECT_PATH=C:\path\to\old\project
   ```

### Сервер (Linux)

1. **Если у вас есть исходные изображения** из старого проекта:
   
   Добавьте в `.env` файл на сервере:
   ```env
   OLD_PROJECT_PATH=/path/to/old/project
   ```
   
   Например:
   ```env
   OLD_PROJECT_PATH=/home/user/old-project
   ```

2. **Если исходных изображений нет** (рекомендуется):
   
   Сидер автоматически создаст placeholder изображения. Просто запустите:
   ```bash
   php artisan db:seed --class=QuizImagesSeeder
   ```

## Автоматический поиск путей

Сидер автоматически проверяет следующие пути:
- `/home/d/dsc23ytp/stroy/public_html` (для вашего сервера)
- `/var/www/html`
- `/home/dsc23ytp/stroy/public_html`
- `C:\OSPanel\domains\lagom` (для локальной Windows разработки)
- `C:\xampp\htdocs\lagom`

## Структура исходных файлов

Если вы указываете `OLD_PROJECT_PATH`, сидер будет искать файлы по следующим путям относительно этого пути:
- `public/img/delete/image-1.png`
- `public/img/delete/image-2.png`
- `public/img/delete/image-3.png`

## Что делает сидер

1. Создает директорию `public/upload/quiz/` (если не существует)
2. Пытается скопировать изображения из старого проекта
3. Если файлы не найдены, создает placeholder изображения с текстом
4. Регистрирует все изображения в медиа-библиотеке

## Результат

После выполнения сидера вы получите 4 изображения:
- `interest-1.jpg` - Купить/продать
- `interest-2.jpg` - Аренда земли
- `interest-3.jpg` - Юридические аспекты
- `interest-4.jpg` - Консультация

Все изображения будут доступны в медиа-библиотеке для использования в квизах.

## Замена placeholder изображений

Если placeholder изображения были созданы, вы можете позже:
1. Загрузить реальные изображения через админ-панель (Медиа)
2. Или запустить сидер снова после настройки `OLD_PROJECT_PATH`

## Примечание

Placeholder изображения имеют размер 245x140 пикселей и цвет фона #657C6C (основной цвет проекта).
