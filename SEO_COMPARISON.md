# Сравнение SEO заголовков: post-ads.ru vs lagom-figma

## Страница /about

### post-ads.ru/about

**Исходный HTML (для ботов):**
```html
<title>Lagom - Профессиональные услуги по работе с земельными участками</title>
```
- ❌ Дефолтный title (главной страницы)
- ❌ Нет специфичного title для страницы About
- ❌ Боты видят одинаковый title на всех страницах

**Динамический (для пользователей):**
- Обновляется через JavaScript (SPA)
- Но боты не видят обновленный title

### lagom-figma.loc/about

**Исходный HTML (для ботов):**
```html
<title>О компании Lagom - Профессиональные услуги по работе с земельными участками</title>
<meta name="description" content="Lagom - ведущая компания по подбору и оформлению земельных участков. Узнайте о нашей команде, опыте работы и подходе к решению задач клиентов. Более 10 лет на рынке недвижимости.">
<meta property="og:title" content="О компании Lagom - Профессиональные услуги по работе с земельными участками">
<meta property="og:description" content="Lagom - ведущая компания по подбору и оформлению земельных участков. Узнайте о нашей команде, опыте работы и подходе к решению задач клиентов. Более 10 лет на рынке недвижимости.">
```
- ✅ Специфичный title для страницы About
- ✅ Правильное описание
- ✅ Open Graph теги
- ✅ Middleware `RenderMetaForBots` заменяет мета-теги для ботов

**Динамический (для пользователей):**
```vue
<SEOHead
    title="О компании Lagom - Профессиональные услуги по работе с земельными участками"
    description="Lagom - ведущая компания по подбору и оформлению земельных участков. Узнайте о нашей команде, опыте работы и подходе к решению задач клиентов. Более 10 лет на рынке недвижимости."
    keywords="о компании, команда, услуги, земельные участки, опыт работы, профессионалы, недвижимость"
/>
```
- ✅ Компонент `SEOHead` обновляет мета-теги при навигации
- ✅ Title и description обновляются для каждой страницы

## Проблема post-ads.ru

1. **Нет серверного рендеринга мета-тегов**
   - Все страницы имеют одинаковый дефолтный title
   - Боты не видят специфичные мета-теги

2. **SEO проблемы**
   - Поисковые системы видят одинаковый title на всех страницах
   - Нет уникальных описаний для каждой страницы
   - Плохая индексация контента

3. **Социальные сети**
   - При шаринге ссылок показывается дефолтный title
   - Нет специфичных Open Graph тегов

## Наше решение

### Middleware для ботов
**Файл:** `app/Http/Middleware/RenderMetaForBots.php`

```php
// Определяет ботов
protected $botAgents = [
    'googlebot', 'bingbot', 'yandexbot',
    'facebookexternalhit', 'twitterbot', 'linkedinbot',
    // ...
];

// Заменяет мета-теги в HTML
$content = preg_replace(
    '/<title>.*?<\/title>/',
    '<title>' . htmlspecialchars($meta['title']) . '</title>',
    $content
);
```

### Vue компонент для пользователей
**Файл:** `resources/js/components/SEOHead.vue`

```vue
<SEOHead
    :title="pageTitle"
    :description="pageDescription"
    :keywords="pageKeywords"
    :og-image="ogImage"
    :canonical="canonicalUrl"
    :schema="schemaData"
/>
```
